# Flow With Market

A premium, single-page trading ebook sales website built with Next.js, Tailwind CSS, Framer Motion, and React Three Fiber.

## Features

- **Modern Design**: Black + Gold + Neon Green theme.
- **3D Elements**: Interactive 3D book mockup using React Three Fiber.
- **Animations**: Smooth scroll and entry animations with Framer Motion.
- **Components**: Built with shadcn/ui for a polished look.
- **Responsive**: Mobile-first design.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the browser**:
    Navigate to [http://localhost:3000](http://localhost:3000).

## Project Structure

- `src/app/page.tsx`: Main landing page assembling all sections.
- `src/components/sections/`: Individual section components (Hero, Pricing, etc.).
- `src/components/ui/`: Reusable UI components from shadcn/ui.
- `src/app/globals.css`: Global styles and theme configuration.

## Customization

- **Theme**: Edit colors in `src/app/globals.css`.
- **Content**: Update text and images in the respective components in `src/components/sections/`.
